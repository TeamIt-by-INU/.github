  ## About

  TeamIt은 팀 빌딩부터 협업, 일정/작업 관리, 화상회의, 분석까지
  캡스톤 프로젝트 진행에 필요한 기능을 통합 제공하는 플랫폼입니다.

  ## Repositories

  - Frontend: TeamIt-by-INU/front (https://github.com/TeamIt-by-INU/front)
  - Backend: TeamIt-by-INU/api (https://github.com/TeamIt-by-INU/api)

  ## Core Features

  - 팀 매칭 및 가입 요청 관리
  - 팀/프로젝트/작업/마일스톤/일정 관리
  - 실시간 채팅 및 화상회의(WebRTC)
  - 신뢰도/협업 분석 대시보드
  - GitHub 연동 기반 협업 지표

  ## Deployment

  - API: GitHub Actions 기반 자동 배포
  - Front: GitHub Actions 기반 자동 배포
